> **Third Year, Second Semester Group Project**
> **Diploma in Information Technology (DIT)**

## 📖 Project Overview

**BookMyStay** is a Django-based web application developed as a group academic project. The system provides an online platform where users can browse available accommodations, make reservations, manage bookings, and securely interact with the system through authentication. Administrators can manage customers, bookings, properties, and other system resources through an administrative dashboard.

This project demonstrates the practical application of Python programming, Django web development, relational databases, version control, and collaborative software engineering.

---

# 🎯 Project Objectives

* Develop a modern web-based booking and e-commerce platform.
* Apply Django's Model-View-Template (MVT) architecture.
* Implement secure authentication and authorization.
* Manage products/services and customer bookings.
* Demonstrate teamwork using Git and GitHub.
* Gain practical experience with full-stack web development.

# 🚀 Running the Project

## Windows

### 1. Clone the repository

```bash
git clone https://github.com/Yessejr/ecommerce.git
```

### 2. Navigate into the project

```bash
cd bookmystay
```

### 3. Create a virtual environment

```bash
python -m venv venv
```

### 4. Activate it

```bash
venv\Scripts\activate
```

### 5. Install dependencies

```bash
pip install -r requirements.txt
```

### 6. Apply database migrations

```bash
python manage.py migrate
```

### 7. Create an administrator account

```bash
python manage.py createsuperuser
```

### 8. Start the development server

```bash
python manage.py runserver
```

Open:

```
http://127.0.0.1:8000/
```
